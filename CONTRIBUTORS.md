## How to Contribute

We welcome contributions to BevyBeats! Here's how to get started:

1. **Fork the repository** - Create your own copy of the project
2. **Set up the development environment** - Follow instructions in the README
3. **Create a branch** - Use a descriptive name for your feature or fix
4. **Make your changes** - Follow the code style and architecture guidelines
5. **Test thoroughly** - Ensure your changes work as expected
6. **Submit a Pull Request** - Provide a clear description of your changes

## Contributor Guidelines

- Follow the existing code style and patterns
- Write clear, descriptive commit messages
- Include tests for new features
- Update documentation as needed
- Be respectful and constructive in comments and discussions
